With managed cloud, Shipfox operates the control plane, including the API,
database, and orchestration. There is no control-plane infrastructure for you to stand up. Setup is
configuration rather than deployment: create a workspace, connect your
[integrations](/integrations/overview), configure [model providers](/reference/model-providers),
and connect [runners](/operations/runners) on compute you control (your jobs still run
on your own runners).

This page will walk through that configuration. For access today, reach out to the
Shipfox team.
