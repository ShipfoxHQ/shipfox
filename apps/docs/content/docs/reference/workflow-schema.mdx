This reference is generated from the strict workflow schema. Workflow files live
at `.shipfox/workflows/*.yml`, where unknown keys are rejected.

<Callout title="Enable editor autocomplete" type="info">
  Add this header to a workflow file:

  ```yaml
  # yaml-language-server: $schema=https://www.shipfox.io/docs/workflow.schema.json
  ```

  In VS Code, you can instead map
  `https://www.shipfox.io/docs/workflow.schema.json` to
  `.shipfox/workflows/*.yml` with `yaml.schemas`.
</Callout>
