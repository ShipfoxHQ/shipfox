The planned Linear integration is intended to receive Linear events in Shipfox.
Its event and agent-tool capabilities are not available.

**Availability:** Coming soon.

## Authentication

Not available.

## Integration connection identity

Linear integration connection slugs are not available.

## Capabilities

| Capability | Availability |
| --- | --- |
| Source control | **Not available.** |
| Events | **Not available.** |
| Agent tools | **Not available.** |

## Available alternative

Use a [custom webhook](/integrations/webhooks) for Linear events today. Create
a webhook integration connection, point a Linear webhook at its ingest URL, and
trigger workflows on `received`. Shipfox passes the request payload to the run
as `event`.
