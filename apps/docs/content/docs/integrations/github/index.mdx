GitHub connects repository work to Shipfox. It provides source control, events,
and agent tools.

## Categories and related terms

**Category:** Source control.

**Related terms:** Git, VCS, CI.

**Availability:** Available.

## Authentication

Shipfox connects through a GitHub App. Install the App on the target account or
organization. Grant it access only to the repositories Shipfox needs.

## Integration connection identity

A GitHub integration connection slug defaults to `github_<account>`, such as
`github_acme`. Use the workspace integration connection slug as a workflow
trigger `source`.

```yaml
triggers:
  on_push:
    source: github_acme
    event: push
```

For trigger field rules, see the [workflow schema reference](/reference/workflow-schema#trigger-fields).
