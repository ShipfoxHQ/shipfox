## Available alternative

Use a [custom webhook](/integrations/webhooks) for Slack events today. Create
a webhook integration connection, point a Slack outgoing webhook or automation
at its ingest URL, and trigger workflows on `received`. Shipfox passes the
request payload to the run as `event`.
